# Misc: WSL

## List the distributions

```powershell
wsl --list
```

## Set the default distribution

```powershell
wsl --setdefault Ubuntu-20.04
```

## Uninstall a distribution

```powershell
wsl --unregister Ubuntu-20.04
```

## Terminate a distribution

```powershell
wslconfig /t Ubuntu-20.04
```
